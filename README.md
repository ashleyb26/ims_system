## Requirements
```
- Java Development Kit (JDK) installed (version 21 or later)
- Maven
```
## Instructions
```
- Run ims_system.sql in migrations folder to add database and table
- Edit application.properties
```
## Start
**Run using maven**
```
$ mvn spring-boot:run
- 127.0.0.1:8080/home
```
**Run using Java**
```
- Click run on folder 'com.example.ims.ImsApplication'
- 127.0.0.1:8080/home
```
